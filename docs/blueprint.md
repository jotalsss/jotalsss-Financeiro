# **App Name**: RealWise

## Core Features:

- Entrada de Renda: Permitir que os usuários insiram suas fontes de renda com valores.
- Entrada de Despesas: Permitir que os usuários insiram suas despesas com categorias e valores.
- Insights com IA: Gerar dicas e sugestões personalizadas com base na entrada financeira do usuário como uma ferramenta útil para melhores decisões financeiras.
- Visão Geral do Painel: Exibir renda, despesas e saldo líquido em um formato visual claro.
- Definição de Metas: Permitir que os usuários definam metas financeiras (por exemplo, economizar para as férias).
- Acompanhamento de Metas: Exibir o progresso em direção às metas.

## Style Guidelines:

- Cor primária: Verde floresta (#4CAF50), evocando crescimento e estabilidade.
- Cor de fundo: Verde claro (#F1F8E9), criando um pano de fundo calmo.
- Cor de destaque: Amarelo suave (#FFEB3B) usado com moderação para chamar a atenção para alertas importantes ou itens acionáveis.
- Fonte limpa e moderna para fácil leitura dos dados financeiros.
- Ícones simples, baseados em linhas, para diferentes categorias de receita e despesa.
- Layout equilibrado com seções claras para renda, despesas, metas e insights baseados em IA.
- Transições suaves e animações sutis para indicar progresso ou mudanças.
